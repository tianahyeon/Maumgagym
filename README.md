## 개요
#### 건강한 라이프스타일에 대한 관심도가 높아지고 있는 요즈음 '오운완'(오늘 운동 완료) 이라는 유행어가 생길 정도로 그만큼 운동은 더이상 빠질 수 없는 필수 취미가 되어가고 있는 것에 착안하여 다양한 운동시설에 대한 정보 및 회원권 서비스를 제공하는 운동 회원권 O2O 서비스 웹사이트를 구현하는 것입니다.
